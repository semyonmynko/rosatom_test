# File Storage FastAPI + SQLAlchemy

1. Create virtualenv & run:

    ```
    $ pip install -r requirements.txt
    ```

2. Run service with command:
    ```
    $ uvicorn app:app --reload
    

3. To run MINIO:
    ```
    $ .\minio.exe server play.min.io
    ```

3. To run tests, execute in terminal:
    ```
    $ cd tests
    $ pytest
    ```
